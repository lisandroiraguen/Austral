# Scripts

This directory contains utility scripts for the Austral project, primarily for minting tokens on the Cardano blockchain using `Lucid` and `@emurgo/cardano-serialization-lib-nodejs`.

## Setup

1.  **Environment Variables**:
    These scripts require a `.env` file in this directory (`scripts/.env`).
    
    Example `.env` content:
    ```env
    BLOCKFROST_PROJECT_ID=preview...
    MNEMONIC=your 24 word mnemonic here...
    NETWORK=Preview
    TOKEN_NAME=Austral-Test
    TOKEN_AMOUNT=100000000
    ```

2.  **Dependencies**:
    Ensure you have installed the project dependencies:
    ```bash
    npm install
    # or
    npm install dotenv tsx @lucid-evolution/lucid @emurgo/cardano-serialization-lib-nodejs
    ```

## Usage

Run the scripts using `npx tsx` from the project root (or inside the scripts folder).

### Mint Tokens

Mints the specified token amount to your wallet, preserving your original Policy ID by using the derived key path `m/1852'/1815'/0'/3/0`.

```bash
npx tsx scripts/mint_token.ts
```

### Check Mnemonic

Validates your mnemonic words against the BIP39 English wordlist. Useful for debugging "Invalid Mnemonic" errors.

```bash
npx tsx scripts/check_mnemonic.ts
```

## Note on Key Derivation

The `mint_token.ts` script uses **manual key derivation** via `@emurgo/cardano-serialization-lib-nodejs` instead of the standard `bip39` library. This is to ensure compatibility with mnemonics generated by certain C# libraries which may have different checksum validation strictness.
